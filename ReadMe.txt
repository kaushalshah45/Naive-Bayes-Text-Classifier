This is a Naive Bayes Text Classifier Java Implementation.

It is based on the bag of words model  and it is frequently used in text categorization.

train.csv: This is the training data. Each row represents a document, each column separated by commas represents
features (word counts). There are 4527 documents and 5180 words.

train-labels.txt: Labels for the training data.

test.csv: Test data, 1806 documents and 5180 words

test-labels.txt: Labels for the test data.
word-indices.txt: Words corresponding to the feature indices.
